# YWC Lego Builder (Lego Block 產生器) — Release v1.8

這是一個為 Blender 設計的輕量級外掛 (Add-on)，專門用來快速產生以 **8mm** 為基準的 LEGO Technic 風格基礎方塊。對於想要在 Blender 內搭建自定義樂高機器人、模擬機構，或是準備 3D 列印自製積木零件的使用者來說，這個工具可以省下大量手動調整尺寸與細分網格 (Loop Cut) 的時間。

## 主要特色

* **標準化尺寸**：完全遵照 1 單位 = 8mm 的樂高/科技件標準。
* **無縫融入工作流程**：直接整合進 Blender 的 `Shift + A` (Add > Mesh) 標準選單中。
* **即時細分網格**：生成的積木會根據您設定的單位數（X, Y, Z）自動在表面生成精確的 8x8mm 網格劃分。不會在內部產生多餘的幾何面，非常適合後續布林運算、挖孔或精確捕捉。
* **原點置底**：積木原點 (Origin) 預設位於底部正中心，方便貼齊網格或是吸附在其他零件的表面上。
* **插入 AXLE (十字孔)**：在編輯模式中選取面，右鍵即可插入十字軸孔。會彈出對話框讓您指定格數（**支援小數**，例如 1.5 格），程式會自動按比例拉長 `AXLE.stl` 後再執行布林切割。
* **插入 PIN (圓柱孔)**：在編輯模式中選取面，右鍵即可插入標準圓孔，自動執行布林運算。
* **插入轉軸 (Shaft)**：在編輯模式中選取面，右鍵選擇插入轉軸孔，會彈出對話框讓您指定格數（**支援小數**，例如 2.5 格），程式會自動按比例拉長 `SHAFT.stl` 後再執行布林切割。
* **自動幾何收尾**：每次「建立積木」或「插入 PIN / AXLE / 轉軸 孔」操作完成後，會自動對結果進行 **消除鬆散點/邊**、**融合共點 (Merge by Distance)**、**重新計算法線 (Recalculate Normals)**，確保網格乾淨可直接用於後續布林或 3D 列印。

## 使用方式

### 建立積木
1. 在 Blender 中按 `Shift + A` > `Mesh` > `Lego Block`。
2. 在左下角「調整上一步操作 (F9)」面板中設定 X、Y、Z 單位數。

### 插入 PIN 孔洞
1. 選取積木物件，按 `Tab` 進入**編輯模式 (Edit Mode)**。
2. 切換為**面選取模式 (按 3 鍵)**。
3. 選取要挖孔的一個或多個 8x8mm 網格面。
4. **按滑鼠右鍵**，在選單底部選擇「**插入 PIN 孔**」。
5. 程式會自動匯入 `PIN.stl` 陰模並執行布林切割。

### 插入 AXLE / 轉軸 (Shaft) 孔洞（支援小數格數）
1. 選取積木物件，按 `Tab` 進入**編輯模式 (Edit Mode)**。
2. 切換為**面選取模式 (按 3 鍵)**。
3. 選取要挖孔的一個或多個 8x8mm 網格面。
4. **按滑鼠右鍵**，在選單底部選擇「**插入 AXLE 孔**」或「**插入 轉軸 孔**」。
5. 在彈出的設定對話框中輸入所需的**格數**（1 格 = 8mm，**可輸入小數**，例如 1.5、2.25、3.75）。
6. 按下「OK」後，程式會將對應的 STL（`AXLE.stl` 或 `SHAFT.stl`）沿深度軸按比例拉長，然後自動執行布林切割。

> **注意**：請確保 `AXLE.stl`、`PIN.stl` 和 `SHAFT.stl` 與 `main.py` 放在同一個資料夾中。

## 更新日誌 (Patch Notes)

### Release v1.8
* **AXLE 選單補上 icon**：插入 AXLE 孔的選單項目原本沒有圖示，現在統一加上 `EMPTY_AXIS` 圖示，與 PIN（`MESH_CYLINDER`）、轉軸（`CON_ROTLIMIT`）視覺一致，比較容易在右鍵選單裡找。
* **移除硬編碼路徑**：`performBooleanCut()` 原本在抓不到 `__file__` 時會 fallback 到 `e:\3D Print\YWC Lego Builder`，這在其他人的電腦上會直接找不到 STL。改為使用 `os.path.abspath(__file__)`，並在 Text Editor 直跑時退回到 `.blend` 檔所在資料夾 (`bpy.path.abspath("//")`)，腳本搬到任何位置都能正常運作。
* **整理 `bl_info`**：作者改為 `YWC Robotics Team (Omni Robotics)`、`blender` 最低版本設為 `(4, 6, 0)`（因為使用了 Blender 4.x 才有的 `wm.stl_import`，並以 4.6 為基準測試），描述也補上 PIN / AXLE / 轉軸 三個編輯模式功能。
* **首個正式 Release 版本**：經過 v1.4 ~ v1.7 數個 BETA 測試後，目前流程已經穩定，正式從 BETA 升格為 Release。

### BETA v1.7
* **自動幾何收尾**：新增 `cleanupMesh()` 共用函式，每次「建立積木」或「插入 PIN / AXLE / 轉軸 孔」執行完畢後，會自動對結果進行：
  1. **融合共點 (Merge by Distance, 0.0001mm)** — 消除布林後在交界產生的重疊頂點。
  2. **消除鬆散點 / 鬆散邊** — 移除沒有連接邊的孤立頂點與沒有連接面的浮動邊。
  3. **重新計算法線 (Recalculate Normals)** — 統一外向法線方向，避免後續陰影/布林破圖。
* 使用者不必再手動切到 Edit Mode 跑 `Mesh → Clean Up → Merge / Delete Loose / Recalculate Normals`，所有積木操作出來的結果都是「即用」狀態。

### BETA v1.6
* **AXLE 新增格數輸入**：插入 AXLE 孔時會彈出「AXLE 設定」對話框，讓使用者輸入想要的格數，程式會自動按比例拉長 `AXLE.stl` 後再執行布林切割，與 轉軸 功能行為一致。
* **格數支援小數**：AXLE 與 轉軸 的「格數」欄位由整數改為浮點數（`FloatProperty`），可輸入任意小數（如 1.5、2.25 格）以對應更精細的長度需求（1 格 = 8mm）。

### BETA v1.5
* **問題修復**: 修復了轉軸功能以4mm爲單位的問題。

### BETA v1.4
* **新增插入轉軸 (Shaft) 功能**：在編輯模式中選取面後，右鍵選單即可插入轉軸孔。點擊後會先彈出「轉軸設定」對話框，讓用戶指定格數（1 格 = 8mm），程式會將 `SHAFT.stl` 沿 Y 軸按比例拉長後再執行布林切割。
* **支援可變長度布林切割**：`performBooleanCut()` 新增 `scaleY` 參數，可在布林運算前對 Cutter 進行自訂縮放，為未來更多可變尺寸零件奠定基礎。

### v1.3
* **新增插入 AXLE 功能**：在編輯模式中選取面後，右鍵選單即可插入標準十字軸孔（自動匯入 `AXLE.stl` 並執行布林運算）。
* **新增插入 PIN 功能**：同上，支援插入標準圓孔（自動匯入 `PIN.stl` 並執行布林運算）。
* **自動 STL 幾何清理**：匯入的 STL 陰模會自動移除重疊頂點、重新計算法線，並微調縮放以避免共面 (Coplanar) 導致的布林運算失敗。
* **自動啟用 STL 匯入外掛**：程式會自動檢查並啟用 `io_mesh_stl`，不需手動開啟。

### v1.1
* **操作邏輯大改版**：將原本寫在側邊欄 (N-Panel) 的介面移除，改為整合進 Blender 官方標準的 `Shift + A > Mesh` 選單中。
* **支援即時參數調整**：生成方塊後，可直接在左下角「調整上一步操作 (F9)」面板中即時拖拉 X、Y、Z 單位數。
* **自動細分功能**：引入了雙線性差值的自訂面生成演算法。現在方塊會依照單位數自動切割好外表面，搭建機器人時更好找參考點。
* **程式碼優化**：變數與類別改用駝峰命名法 (camelCase)，並將程式碼內的註解翻新為更直白的人類對話口吻。

### v1.0
* 初始版本發布。
* 具備最基礎的任意尺寸無凸起積木生成功能。
