# YWC Lego Builder (Lego Block 產生器)

這是一個為 Blender 設計的輕量級外掛 (Add-on)，專門用來快速產生以 **8mm** 為基準的 LEGO Technic 風格基礎方塊。對於想要在 Blender 內搭建自定義樂高機器人、模擬機構，或是準備 3D 列印自製積木零件的使用者來說，這個工具可以省下大量手動調整尺寸與細分網格 (Loop Cut) 的時間。

## 主要特色

* **標準化尺寸**：完全遵照 1 單位 = 8mm 的樂高/科技件標準。
* **無縫融入工作流程**：直接整合進 Blender 的 `Shift + A` (Add > Mesh) 標準選單中。
* **即時細分網格**：生成的積木會根據您設定的單位數（X, Y, Z）自動在表面生成精確的 8x8mm 網格劃分。不會在內部產生多餘的幾何面，非常適合後續布林運算、挖孔或精確捕捉。
* **原點置底**：積木原點 (Origin) 預設位於底部正中心，方便貼齊網格或是吸附在其他零件的表面上。
* **插入 AXLE (十字孔)**：在編輯模式中選取面，右鍵即可插入標準十字軸孔，自動執行布林運算。
* **插入 PIN (圓柱孔)**：在編輯模式中選取面，右鍵即可插入標準圓孔，自動執行布林運算。

## 使用方式

### 建立積木
1. 在 Blender 中按 `Shift + A` > `Mesh` > `Lego Block`。
2. 在左下角「調整上一步操作 (F9)」面板中設定 X、Y、Z 單位數。

### 插入 AXLE / PIN 孔洞
1. 選取積木物件，按 `Tab` 進入**編輯模式 (Edit Mode)**。
2. 切換為**面選取模式 (按 3 鍵)**。
3. 選取要挖孔的一個或多個 8x8mm 網格面。
4. **按滑鼠右鍵**，在選單底部選擇「**插入 AXLE 孔**」或「**插入 PIN 孔**」。
5. 程式會自動匯入對應的 STL 陰模並執行布林切割。

> **注意**：請確保 `AXLE.stl` 和 `PIN.stl` 與 `main.py` 放在同一個資料夾中。

## 更新日誌 (Patch Notes)

### v1.3
* **新增插入 AXLE 功能**：在編輯模式中選取面後，右鍵選單即可插入標準十字軸孔（自動匯入 `AXLE.stl` 並執行布林運算）。
* **新增插入 PIN 功能**：同上，支援插入標準圓孔（自動匯入 `PIN.stl` 並執行布林運算）。
* **自動 STL 幾何清理**：匯入的 STL 陰模會自動移除重疊頂點、重新計算法線，並微調縮放以避免共面 (Coplanar) 導致的布林運算失敗。
* **自動啟用 STL 匯入外掛**：程式會自動檢查並啟用 `io_mesh_stl`，不需手動開啟。

### v1.1
* **操作邏輯大改版**：將原本寫在側邊欄 (N-Panel) 的介面移除，改為整合進 Blender 官方標準的 `Shift + A > Mesh` 選單中。
* **支援即時參數調整**：生成方塊後，可直接在左下角「調整上一步操作 (F9)」面板中即時拖拉 X、Y、Z 單位數。
* **自動細分功能**：引入了雙線性差值的自訂面生成演算法。現在方塊會依照單位數自動切割好外表面，搭建機器人時更好找參考點。
* **程式碼優化**：變數與類別改用駝峰命名法 (camelCase)，並將程式碼內的註解翻新為更直白的人類對話口吻。

### v1.0
* 初始版本發布。
* 具備最基礎的任意尺寸無凸起積木生成功能。
